# ПОЕЗДУН ака МАТЕРШИНИК

v 1.0.0  
04.07.26

Телеграм бот, который реагирует на сообщения с неправильно написаной нецензурной лексикой, отправляя в ответ видео.
Вдохновлён шуточной командой sl для линукс и юникс систем, которая выводит анимацию поезда в терминале при ошибочном вводе команды ls.

Контрибьюторы: Tema Normalny (@Tema_Normalny on tg), //:DAVIDHAIT (@DAVIDHAIT on tg), iquldev (@iquldev on tg).s  
Бот был создан для чата тиктокера МишШиш414 - РАЙ.

Для работы бота необходимо установить зависимости.
Это можно сделать с помощью pip: pip install -r requirements.txt
Также, если вы пользователь NixOS, можно использовать команду nix-shell -p python310Packages.aiogram для установки aiogram,
или использовать файл shell.nix прикреплённый к репозиторию, чтобы создать среду с установленной библиотекой aiogram.
После установки не забудьте прописать свои данные в файл .env (токен от @BotFather и путь до видео).

Если хотите помочь с разработкой бота, можете написать мне в телеграм - @DAVIDHAIT
(...Да, бот был частично создан на NixOS, всем советую :D)

# СОДДЕРЖАНИЕ

В репо несколько файлов. Сам код на пайтон, видео с поездом, флейк для пользователей NixOS, а также файлы .env для настроек, words.txt со словами и скрипты для запуска.

В коде также присутсвует полторы посхалки

# ПРЕДНАЗНАЧЕНИЕ

Это лишь шуточный бот. однако...
Его совсем не сложно модифицировать под себя, даже не имея никакого опыта на пайтон. Взял, поменял видео, поменял список слов в words.txt, да и всё!

САААМОЕ ВАЖНОЕ! чтоб превратить этот код в бота в телеграме, зайтите в бота BotFather, создайте там бота и вставьте полученный токен в файл .env (в поле BOT_TOKEN). Также можете поменять путь к видео (VIDEO_PATH) и слова-триггеры в файле words.txt. После чего запускаете скрипт START.sh (Linux/macOS) или START.bat (Windows).
